Finish the IngredientsComponent and CakeComponent component.

The IngredientsComponent components will receive an array of strings as the ingredients array. It should render one li element per item in the array.

For example, if the ingredients array is ["Bread", "Eggs", "Milk"], the rendered HTML should look like:

<ul>
    <li>Bread</li>
    <li>Eggs</li>
    <li>Milk</li>
</ul>
